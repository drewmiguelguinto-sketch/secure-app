# Secure App - Week 15 Iteration

This repository demonstrates a DevSecOps CI/CD pipeline using GitHub Actions.

## Features
- Secure Python app with pinned dependencies
- Dockerfile with best practices
- SAST, dependency, secret, and container scans
- SBOM generation (CycloneDX)
- Artifact storage (scan results, SBOM, vulnerability report, trusted build marker)
- Multi-job pipeline with enforced dependencies
